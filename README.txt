El metodo procesamiento con una pila fue más efectivo en este ejercicio que el de una cola. La razón principal es que las pilas son estructuras de datos LIFO (último en entrar, primero en salir), lo que significa que el último elemento agregado es el primero en ser extraído, mientras que las colas son estructuras de datos FIFO (primero en entrar, primero en salir), lo que significa que el primer elemento agregado es el primero en ser extraído.
